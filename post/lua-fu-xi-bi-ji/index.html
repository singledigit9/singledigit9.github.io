Lua复习笔记

读书笔记 Lua

简介

关于Lua的复习整理。方便以后查找&避免遗忘。

基本介绍

变量与类型

  • Lua是动态语言，变量没有类型，但值有类型。
  • 所有的值都是 第一类对象
  • 8种基本类型：
    • nil：任何未赋值的变量均为nil
    • boolean：nil和false在判断中均视为false，其他值视为true
    • number：整数和浮点数
    • string：可以视为不可改变的byte队列。
    • function：函数。是对象。
    • userdata：是对象。
    • thread：线程，是对象。
    • table：是对象。除了nil和 NaN 不能作为索引，其他的所有值均能作为索引。是Lua中唯一的数据结构机制。如果使用number索引，浮点数如果能够转换成整数的话，则索引值默认改成对应的整数，eg. a[2.0]与a[2]

元表和元方法(Metatables and Metamethods)

  • 任何值都有元表
  • 元方法定义了值的基本行为，如加减等操作。
  • 可以使用getmetatable来获取元表；使用setmetatable替换元表
  • 每个table和userdata有自己单独的元表，复合的table和userdata共享元表。其他的值每种类型共享一个元表。
  • 当外部触发调用原方法时，先在当前的元表中查找对应的元方法，没有则在上一级元表中继续查找，直至找到对应的元方法或者查找完所有的元表。找到则执行返回结果，找不到则报错。
  • 所有的元方法以及其含义：
    • __add : +
    • __sub : -
    • __mul : *
    • __div : /
    • __mod : %
    • __pow : ^
    • __unm : 取反
    • __idiv : // 等价于 (int)(a/b)
    • __babd : 按位与
    • __bor : 按位或
    • __bxor : 按位异或
    • __bnot : 按位取反
    • __shl : 按位左移
    • __shr : 按位右移
    • __cancat : .. 字符拼接
    • __len : 默认情况下，返回字符串或者table的长度
    • __eq : ==比较等于
    • __lt : <比较小于
    • __le : <=比较小于等于
    • __index : a[b] 索引访问操作执行。当目标不是table或者table中没有对应索引时触发。
    • __newindex : a[b] = c 索引赋值才做执行。当目标不是table或者table中没有对应索引时触发。
    • __call : 非function作为函数调用时触发，惟一一个能够返回多个值的元方法。
    • __gc : table和userdata 执行垃圾回收时调用。

垃圾回收(GC)

  • Lua中的内存是自动分配的，gc也是自动的。当一个对象的引用为0时，将其加入回收列表，等待回收。
  • 特殊的table：弱表(weak table)
    • 描述：弱表是一个弱引用的表。
    • 定义：重新设置table的元表，添加__mode字段。其中字段值等于"k"表示key是弱引用，"v"表示值是弱引用，"kv"表示索引和值都是弱引用。
    weakTable = {"xxx", "yyy", "zzz"};
setmetatable(weakTable, {__mode = "v"})

协同程序(Continues)

  • 在lua中，协程也相当于是独立的线程，但和线程不同的是，协程只能通过显示调用和yield函数来挂起和执行。
  • 激活执行流程：1.创建协程；2.激活协程；3.执行到 任意coroutine.yield返回；4.再次激活；5.接着上次未执行的地方继续执行，遇到coroutine.yield返回。（任意 表示函数中嵌套的函数里如果有coroutine.yield的话，一样跳出返回）
  • 协程相关的函数
    • coroutine.create([function])：创建协程，返回thread类型。该方法并不启动协程。
    • coroutine.resume([thread],[arg...])：激活协程，协程结束或者yield结束此次运行。如果中间出错，则会终止该协程，但不会终止程序。
      • 参数：[要激活的协程],[可选传递参数]。
      • 返回：true, [coroutine.yield(xxx)中返回的参数] 或者 false,错误信息。
    • coroutine.yield([arg...])：协程跳出。每一次resume，执行到一个yield；下一次resume，接着未执行的代码继续执行。
    • coroutine.wrap：创建协程。
      • 返回：function。
      • 使用说明：当外部调用返回的函数时，激活对应的协程，并有返回值。其返回值和resume相比除了没有第一个bool返回值，其余相同。并且，当函数中运行出错时，会抛出异常，中断程序。
  • 官网的协程的示例代码
    • 代码：
    function foo (a)
	print("foo", a)
	return coroutine.yield(2*a)
end

co = coroutine.create(function (a,b)
	print("co-body", a, b)
	local r = foo(a+1)
	print("co-body", r)
	local r, s = coroutine.yield(a+b, a-b)
	print("co-body", r, s)
	return b, "end"
end)

print("main", coroutine.resume(co, 1, 10))
print("main", coroutine.resume(co, "r"))
print("main", coroutine.resume(co, "x", "y"))
print("main", coroutine.resume(co, "x", "y"))
    • 返回值
    co-body 1       10
foo     2
main    true    4
co-body r
main    true    11      -9
co-body x       y
main    true    10      end
main    false   cannot resume dead coroutine

语法与常用函数

常用

  • 输出：print("xxxx")
  • 注释：
    • 单行注释：--
    • 多行注释：
    --[[
    多行注释
--]]
  • 命名：
    • 大小写敏感
    • 尽量不要用下划线加大写字母的组合，避免和Lua内置的命名冲突
    • 其余规则和其他语言基本一致
  • 任何位置 声明的变量默认为全局变量，未声明的变量为nil
  • 多重赋值：
    • 示例：
    a = 5
b = 6
a,b = b,a
    • 说明：要赋值的变量多时，多余的变量用nil补足；要赋值的变量少时，多余的值舍弃不用。
  • local变量的作用范围：
    • 同一个区块内
    • 示例：下面的输出为 nil nil
    function A()
    local a = 5;
end

do
    local b = 6;
end

print(a, b)
  • 模块加载：require("xxxxx")

基本类型的声明和使用

  • nil：

    • 对于全局变量和table，令其等于nil，等同于将其删除
    • 判断一个变量是否为 nil的方法为：type(x) == "nil"。需要加上双引号。

  • boolean：

    • false和nil在if判断中均为false

  • number：（默认是double）

    • 科学计数法：[X]e[Y] === X * 10^Y。eg. 2e2 = 2 * 10^2 = 200

  • string：

    • [[ …… ]]：表示一块字符串，可以跨行。
    • 字符串连接：..
    • 数字字符串，数字之间进行算术操作时，字符串会自动转换成对应的数字进行运算。如果转换失败，则会报错。eg. “2” + 6 = 8
    • 转义字符：下面记录一些常用的转义字符
      • \n：
      • \r：
      • \t：横向制表符
      • \v：纵向制表符
      • \：
      • '：
      • "：
      • \0：空字符
    • 字符串操作：
      • string.upper(target)
      • string.lower(target)
      • string.gsub(target, find, replace, num)：字符串替换。
        • 参数说明：目标字符串， 要查找的字符串，要替换成的字符串，替换的次数(省略表示替换所有)
        • 返回值：[替换后的字符串], [替换的次数]
      • string.find(target, find, index)：字符串查找
        • 参数说明：目标字符串， 要查找的字符串， 查找的起始位置
        • 返回值：如果找得到： [找到的字符窜的开始位置],[找到的字符串的结束位置]；如果找不到：nil
      • string.reverse(target)：反转字符
      • string.format(pattern, ...)：字符串格式化
        • 参数说明：格式字符串， 填充到格式字符串中的内容
      • string.char(...)：将整数转成对应的字符并拼接
      • string.byte(target, index)：将字符转成对应的整数值。索引-1表示倒数第一个字符
        • 参数说明：目标字符串，字符串索引值。第二个参数可以省略，默认为1
      • string.len(target)：获取目标字符串的长度
      • string.rep(target, n)：将目标字符串复制n次，并拼接
      • .. : 连接两个字符
      • string.gmatch(target, pattern)：迭代器函数，每一次调用这个函数，都会返回符合格式的字符串，如果没有找到，则返回nil
        • 参数说明：目标字符串， 格式字符串
        • 示例：
        for word in string.gmatch("Hello Lua user", "%a+") do
    print(word)
end
      • string.match(target, pattern, index)：返回符合格式的第一个字符串
        • 参数说明：目标字符串， 格式字符串， 搜索开始位置
    • 格式化字符串规则：
      • format
        • 基本规则：
          • %c：数字取整转换成Ascii码中对应的字符
          • %d, %i：数字取整
          • %o：数字取整转八进制
          • %u：将大于-1的数强制转成整数。如果输入的数字小于等于-1， 则会报错
          • %x：数字取整转十六进制，使用小写字母
          • %X：数字取整转十六进制，使用大写字母
          • %e：数字转为科学计数法，使用小写字母e
          • %E：数字转为科学计数法， 使用大写字母E
          • %f：数字转为浮点数表示
          • %g(%G)：数字绝对值的大于0且整数部分数字个数大于6时，同%e(%E)；数字绝对值小于0或者整数部分数字个数小于等于6时，显示6位有效数字，其余舍弃。
            • eg. 123456.123456 --> 123456 ; 1234567 --> 1.23457e+06 ; 0.0123123123 --> 0.0123123
          • %q：将string转化成lua中可以安全读取的模式。 会将字符进行一些转换，具体规则带查明
          • %s：接受字符串，并按照给定格式输出。
        • 进阶规则：
          • 可以在%后面添加参数
            1. 符号
            2. 占位符
            3. 对其标识
            4. 宽度数值
            5. 小数位数/字符裁切
          • 示例
            • %04d：数值取整， 不足4位数字用0补全
            • %+d：数值为正数时，显示数字“+”号
            • %.4f：输入数值保留4位小数
            • %3s：输入字符串长度小于3时，在字符串 左面 补足空格，使字符串长度等于3。字符串长度大于等于3时直接输出字符串。
            • %-3s：作用同上，不同的是在字符串 右边 补足空格
            • %5.3s：截取字符串中前三个字符，并用空格在左侧补全至5个字符
            • %-5.3s：作用同上，但补全的空格从右侧开始
      • 正则表达式
        • 基本规则：
          • 单个字符(除 ^ $ ( ) % . [ ] * + - ? 外): 与该字符自身配对
          • .(点): 与任何字符配对
          • %a: 与任何字母配对
          • %c: 与任何控制符配对(例如\n)
          • %d: 与任何数字配对
          • %l: 与任何小写字母配对
          • %p: 与任何标点(punctuation)配对
          • %s: 与空白字符配对
          • %u: 与任何大写字母配对
          • %w: 与任何字母/数字配对
          • %x: 与任何十六进制数配对
          • %z: 与任何代表0的字符配对
          • %x(此处x是非字母非数字字符): 与字符x配对. 主要用来处理表达式中有功能的字符( ^ $ ( ) % . [ ] * + - ? )的配对问题, 例如%%与%配对
          • 上面的匹配字符，如果%后面的字母是大写的，则表示除了该类型字符不匹配其余都匹配。例如：%A与任何非字母匹配

  • table：

    • 声明：

    • 索引
      • 列表：与其他语言索引从0开始不同，lua中索引从1开始
      • 字典：对于字符索引，可以有以下方式取值赋值
      t["key"] = "value";
t.key = value;
    • 获取长度：
      • #(t)：长度识别从索引1开始，如果没有索引1，则返回0；之后顺序查找连续索引，直至最大值。eg. t[1] = 1; t[2] = 2; t[4] = 4; 返回的长度为2
      • table.maxn(t)：获取的是table的最大整数数字索引。上面的例子中的table，如果使用这个函数，则返回4。
    • Table操作
      • table.concat(target, sep, start, end)：输出全部table元素字符串
        • 参数说明：目标table，分隔符（可省略），开始索引（可省略）， 结束索引（可省略）
      • table.insert(target, index, value)：插入元素
        • 参数说明：目标table，插入位置（可省略，默认为最后），插入元素
      • table.remove(target, index)：
        • 参数说明：目标table，移除的位置（可省略，默认为最后一个）
      • table.sort(target, compareFunction)：
        • 参数说明：目标table， 比较函数（可省略，省略后按默认内置排序进行比较）

  • function：

    • 示例：
    function Add(a, b)
    return a + b;
end
    • 函数默认声明为全局函数。标明为local时，声明的函数为局部函数
    • 多返回值，使用return a, b, c， 返回值的使用同多重赋值
    • 不定参数：
      • 示例
      local function A(...)
    local args = {...}
end
      • 获取不定参数长度
        • 将不定参数存储到table中，获取table列表的长度#{...}
        • 使用select函数。select("#", ...)
      • select函数的说明
        • select("#", ...)：返回不定参数的长度。
        • select(n, ...)：返回第n个参数值

流程控制

判断

  • 示例：
if (a < 0) then

elseif (a == 0) then

else

end
  • 特殊说明：除了false和nil是假，其余都为真。(if判断中0为真)

循环

  • while循环
a=10
while( a < 20 )
do
   print("a 的值为:", a)
   a = a+1
end
  • for循环
    • 示例：
    for i = 1, 10, 2 do
    print(i)
end
    • 说明：起点值，终点值，步长。其中步长可以省略，默认为1。
    • 遍历：
    for k, v in pairs(t) do     --按照hash顺序遍历，而不是table中值的添加顺序
    print(k .. ":" .. v);
end

for k, v in ipairs(t) do    --按照索引从1开始递增遍历，索引有跳跃时会停止。
    print(k .. ":" .. v);
end

for i=1, #(t) do            --从1开始，获取table中最大连续索引值
    print(t[i])
end
  • repeat...until：
    • 示例：
    local a = 10
repeat
    print("a的值为:", a)
    a = a + 1
until (a > 15)

文件IO

  • io.open(path, mode)：文件打开/创建，返回file对象
    • 参数说明：目标路径，打开模式（可省略）
    • 模式参数以及其意义
      • r：只读打开，文件必须存在
      • w：只写打开。文件不存在则创建，文件存在则清空文件再打开。
      • a：只写打开。文件不存在则创建，文件存在则打开，新写入的数据将添加到原数据的末尾。（源文件的EOF符号保留）
      • b：二进制模式。如果文件是二进制文件，可以加上b。
      • +：表示可读可写。
      • 可以用这种方式[r/w/a][b][+]进行组合。
  • io.close(file)：关闭已打开的file
  • io.input(file)：设置默认的操作file
  • io.read(mode)/file.read(mode)：读取已打开文件内容
    • 参数说明：模式，可省略，默认为第一行文件

注释

  1. 第一类对象：指程序中所有的实体。可以存入变量或结构；可以作为参数传递；可以作为返回值；可以执行时创建；有 固定身份
  2. 固定身份：指实体有内部表示，而不是依靠名字来进行识别，比如匿名函数
  3. NaN：一个特殊值，表示未定义或者运算无效的返回值。
  4. 弱引用：弱引用时不计入引用计数。即gc会忽略弱引用。
