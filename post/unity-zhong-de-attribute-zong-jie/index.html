Unity中的Attribute总结

2019-01-04 · 3 min read
Unity3D

简介

对Unity中的[xxx] 常用属性进行了一个总结。
有部分基本用不上 || 使用方法不明的 暂时没有添加进来。

正文

  • 通用 :
    • 作用于类
      • AddComponentMenu : 添加到Component菜单，使脚本可以作为Component添加
      • DisallowMultipleComponent : 不允许在同一物体上添加多个该组件
      • ExecuteInEditMode : 该脚本的可以在编辑器模式下运行。(OnGUI:仅gameview接收事件;Update:仅scene中物体变化)
      • RequireComponent : 添加脚本时检查物体有没有对应组件，没有则添加
      • SharedBetweenAnimators : 多个Animation共享一个StateMachineBehaviour实例，减小内存占用。仅适用于StateMachineBehaviour类
      • CreateAssetMenu : 能够在Create中创建，仅适用于ScriptableObject类
      • ImageEffectAllowedInSceneView : 相机特效同样在scene中生效
      • Serializable : 声明序列化
      • HelpURL : 点击脚本上帮助按钮跳转的链接。（网址注意添加“https://”）
      • SelectionBase : scene中选择物体时，如果其父物体上挂有这个脚本，则改为选择该父物体，而不是选择它自己。
    • 作用于方法
      • ContextMenu : Inspector面板中脚本对应的"设置"界面中添加选项
      • ImageEffectOpaque : 仅用于OnRenderImage方法上，使该渲染顺序变为:非透明物体，"标签渲染"，透明物体
      • RPC : 网络通信中对方法进行RPC调用
      • RuntimeInitializeOnLoadMethod : 游戏启动时调用该静态方法
    • 作用于变量
      • ContextMenuItem : Inspector面板中脚本对应的变量上添加右键菜单，非公有变量需要额外序列化
      • Header : Inspector面板上显示的变量分组
      • HideInInspector : Inspector面板隐藏公有变量
      • Multiline : 使string的编辑区域可以多行编辑，非公有变量需要额外序列化
      • Range : 限定编辑器数值范围，非公有变量需要额外序列化
      • SerializeField : 强制序列化
      • Space : 在Inspector面板上变量之间插入空行
      • TextArea : 在Inspector面板string变量的输入变成TextArea
      • Tooltip : 鼠标Hover的提示文字
      • elayed : 确认修改后才修改生效，非公有变量需要额外序列化
  • 仅编辑器:
    • 作用于类:
      • CanEditMultipleObjects : 该编辑脚本所对应的编辑内容可以多选编辑
      • CustomEditor : 标记该类为哪个类型的编辑器
      • CustomPreviewAttribute : 标记该类为某个类型的自定义预览
      • CustomPropertyDrawer : 标记该类为某类型/结构的自定义属性显示，仅适用于PropertyDrawer
      • InitializeOnLoad : Unity启动时运行该类，需要静态构造函数
      • PreferenceItem : 自定义 Unity Preferences 额外的标签面板
    • 作用于方法
      • InitializeOnLoadMethod : Unity启动时执行该方法，需要静态方法
      • MenuItem : 菜单上添加选项，需要静态方法
      • DrawGizmo : 自定义Scene视图中的Gizmo，需要静态方法
      • PostProcessBuild : 打包完成后调用静态方法
      • PostProcessScene : 打包完场景是调用对应静态方法
